# AGENTS for CS50 Workspace

This workspace is a personal CS50 practice repository containing small assignment files in C and Python.

## What this repo contains
- Simple console programs and exercises.
- Primary languages: `C` and `Python`.
- Files are typically named by assignment (e.g. `exercise2.c`, `exercise9.py`, `hello.c`).
- There is no formal build system or tests checked in.

## How to help
- Prefer minimal, targeted edits to the specific exercise file the user mentions.
- Preserve the current program structure and user intent.
- For C files, assume standard compilation with `gcc`/`clang` and `stdio.h`.
- For Python files, assume Python 3 and console input/output.
- Fix syntax and logic issues, but do not rewrite files into a different assignment format unless asked.

## Typical tasks
- Correct syntax errors.
- Fix off-by-one and conditional logic bugs.
- Improve input/output prompts and validation for simple exercises.
- Keep changes small and review the file name to ensure the right assignment is modified.

## Notes for agents
- Do not invent project scaffolding or new directories; this repo is a flat exercise collection.
- There are no shared libraries or modules to import across exercises.
- If the user asks for a new feature, make it consistent with the exercise style and keep it self-contained.
